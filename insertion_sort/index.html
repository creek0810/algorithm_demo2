Name: Insertion sort

Time complexity: O(x2)

Space complexity: Θ(1)

Data structure: Array

Detail: Stable sort

Code

void insertion_sort(vector&array){ for(int i = 1; i < array.size(); i++){ int current = array[i]; int j = i - 1; while(j >= 0 && array[j] > current){ array[j+1] = array[j]; j--; } array[j+1] = current; } }
